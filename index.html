Some basic features we have included in the Google_S Starter Theme

  • Various sizes for the thumbnails.
  • A graphic grid for Media Galleries.
  • A custom field to insert text before the image featured in posts and pages.
  • A Right Sidebar fully responsive.
  • A Footer Sidebar with default content.
  • Lightbox for images with link media and slide for tunnels.
  • Simple jquery dropdown sub-menu.
